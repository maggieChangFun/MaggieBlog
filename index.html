【小工具】測試工具 - postman

發表於 2019-01-16

前言

最近在寫給手機介接的API，利用 postman 來測試，
所以就來記錄一下我所使用到的postman的方法。

POST

方法一

“Body” 中選擇 “x-www-form-urlencoded” ，再將參數加入下方輸入格。
Headers 中 Content-Type 的value會變成 application/x-www-form-urlencoded。

補充：x-www-form-urlencoded
  • POST 提交數據的方法之一。
  • 數據按照 key1=val1&key2=val2 的方式進行編碼，key與value都進行了url轉碼。
插曲：

一開始在使用postman時有選擇post,但一直把參數寫在Params。因此傳base64時，後端decode一直失敗。

方法二

“Body” 中選擇 “raw” 之後 ，text 選擇 “JSON (application/json)”，在下方以json格式加入參數。Headers 中 Content-Type 的value會變成application/json。

補充：application/json
  • 以json格式提交參數。

「swift 學習筆記＆心得」02 - 使用playgrounds 與學習swift基礎語法

發表於 2019-01-04

開發ios APP必學項目

  1. swift：開發 iosApp 的程式語言。
  2. Xcode10：開發工具。
  3. iOS software development kit：apple提供的軟體開發工具，可以加速開發。

playground

Swift 程式語言的互動開發環境，能即時得到運行結果。

語法與備註

var & let

var 是用來宣告變數，let是用來宣告常數。
常數是以一個值來做初始化，不能改變。
變數是可以改變的。

Type Inference

宣告常數與變數時可以省略型態，swift編譯器會自己判斷。
宣告的語法 ：var number: Double = 10.5
名稱後加 : 型態

text

宣告字串語法：String a = "test"
可以使用 + 串連兩個相同型態的元素,
始用uppercased() 與 lowercased()改變大小寫,
利用count計算字元數目。

型態轉換

型態轉換語法：欲轉換的型態（要轉換的變數）
String(ten)

字串插值（String Interpolations）

字串插值語法：$ \ (要轉換的變數)
var totalPriceMessage = "The price of the book is $ \(totalPrice)"

一直都找不到我願意堅持記帳的app，所以就很想要自己來寫一個。（可能也是我自己的問題？！哈哈）
